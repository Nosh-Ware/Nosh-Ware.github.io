RPG Setting Editor


What is it?


Its a small editor used to modify the settings not normally editable directly in the original editor. I should be noted that these setting can be easily edited by open some of the files in you project with notepad (or whatever basic text editor your OS uses), this just serves to act as a slightly easier means to edit it.
